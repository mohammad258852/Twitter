# Twitter

a simple terminal-based twitter app.

## run

```bash
sudo apt install make libncurses5
git clone https://github.com/mohammad258852/Twitter
cd Twitter
```
### run server
cd server
make main
./main

### run client
cd client
make main
./main

## feature
1.  send and get tweets
2.  comment and like tweets
3.  follow users
4.  delete tweets
5.  retweet tweets
6.  set bio and see others bio
7.  ...
